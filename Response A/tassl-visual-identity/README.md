# TASSL Visual Identity Web App

تطبيق Vite + React + TypeScript + Tailwind CSS يعرض دليلاً تفاعلياً لهوية بصرية مستلهمة من الصورة المرفقة لشعار TASSL.

## ما يحتويه التطبيق

- صفحة عربية RTL كاملة.
- عرض الصورة الأصلية المستخدمة كمرجع.
- جوهر الهوية: الهندسة، الثقة، الطبيعة، والوجهة.
- نظام شعار مقترح مع رمز سقف ذهبي.
- لوحة ألوان مستخرجة من الصورة مع أكواد HEX/RGB وقابلية نسخ اللون.
- نظام طباعي وباترن تضاريس مستوحى من الخلفية.
- تطبيقات عملية: بطاقة عمل، قوالب، استخدامات مطبوعة ورقمية.

## التشغيل

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

النسخة المبنية موجودة في `dist/` بعد تنفيذ البناء.
